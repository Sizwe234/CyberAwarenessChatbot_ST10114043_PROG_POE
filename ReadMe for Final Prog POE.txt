CyberAwarenessChatbot - Final POE =================================

Overview:

Cy-Bot is a Windows Forms cybersecurity awareness chatbot designed to educate users on safe online practices. It features:

Natural language understanding with simulated NLP

Keyword recognition and sentiment detection

Task assistant with reminders

Cybersecurity quiz game

Activity log of all user interactions

Voice greeting and ASCII console branding

Features:

Part 1: Audio intro, ASCII art, keyword matching

Part 2: Sentiment detection, varied responses, memory of interests

Part 3: Windows Forms GUI, Task Manager, Quiz Game, Activity Log, NLP recognition
